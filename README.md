# Nixie
<img width="433" height="433" alt="image" src="https://github.com/user-attachments/assets/fad904ef-bc21-4354-a2b8-9dc044d9660e" />

Nixie e uma assistente pessoal/IA model em desenvolvimento.

Este repositorio publico existe como uma vitrine do projeto: um lugar para reunir prints, registros visuais, ideias, evolucao, roadmap e demonstracoes de como a Nixie esta ficando ao longo do tempo.

> Este repositorio nao representa o codigo completo, nem documenta a arquitetura interna do projeto.

## Estado atual

A Nixie esta em fase experimental, com foco em:

- personalidade e identidade visual;
- interface de interacao;
- modelo 3D e expressoes;
- memoria, contexto e comportamento;
- testes de voz, animacao e presenca;
- demonstracoes visuais do progresso.

O objetivo deste repositorio e mostrar a evolucao publica da Nixie sem expor partes sensiveis, privadas ou incompletas do desenvolvimento.

## Prints e demonstracoes

### Modulagem

Novos prints, telas e demonstracoes visuais serao adicionados aqui conforme o projeto avancar.

## Sobre o projeto

Nixie e um projeto pessoal de IA com identidade propria, pensado como uma assistente digital com presenca, memoria, visual 3D e interacoes mais naturais.

Este espaco serve principalmente para acompanhar como ela esta atualmente, registrar ideias e organizar materiais publicos de apresentacao.

## Roadmap publico

- Organizar mais prints da interface atual.
- Adicionar demonstracoes visuais do modelo 3D.
- Mostrar evolucoes de tela, voz e comportamento.
- Registrar ideias futuras sem expor detalhes internos.
- Melhorar a documentacao visual do projeto.

## Aviso

Este repositorio e apenas para apresentacao, documentacao visual e acompanhamento publico.

Nem todo conteudo do projeto esta disponivel aqui. Partes internas, codigo principal, prompts, fluxos, configuracoes, modelos, memorias e arquivos privados podem estar omitidos ou protegidos.
